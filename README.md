

# ​​ Local Food Wastage Management System

An interactive **Streamlit dashboard** powered by **Python & SQLite**, designed to minimize food wastage by efficiently connecting surplus food providers with receivers in need. The system offers data-driven insights, streamlined claims management, and seamless redistribution.

##  Project Overview

Surplus food often goes to waste while individuals and NGOs struggle with food access. This project tackles that gap through:

- **Food Listings**: Providers can list surplus food items with details like expiry date, location, and meal type.
- **Claim Management**: Receivers can claim available listings, while the system tracks claim status (Completed, Pending, Cancelled).
- **Analytics Dashboard**: Built with Streamlit, it provides rich visual insights into providers, receivers, meal preferences, city-wise trends, and more.

###  Objectives

- Reduce food wastage through effective redistribution.
- Enable transparent, data-driven matching of food supply and demand.
- Provide actionable insights for better strategic planning and social impact.

##  Setup Instructions

1. **Clone this repository**

```

git clone [https://github.com/Pavan-Kumar-Dirisala/Local-Food-Wastage-Management-System.git](https://github.com/Pavan-Kumar-Dirisala/Local-Food-Wastage-Management-System.git)

```

2. **Install dependencies**

```

pip install -r requirements.txt

```

3. **Run the application**

```

streamlit run app.py

```

Open the provided localhost URL in your browser to start using the app.
